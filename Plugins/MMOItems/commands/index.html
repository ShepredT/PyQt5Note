    Commands

    ITEM COMMANDS
    /mmoitems create Creates a new item with the specified item type and item ID. ID is NOT the display name.
    /mi copy Copies the item you are holding into a new item with a different ID. Useful if mass producing.
    /mi delete Deletes the specified item.
    /mi edit Opens up the item editor.
    /updateitem Updates the item you are holding.
    /updateitem Turns on the automatic Item-Updater for a specific item. It is not totally recommended to leave this on constantly as always updating items may eventually cause lag if you have enough.
    /mi drop Generates and drops (essentially spawn in an item without putting it in a player inventory).
    /mi item identify Manually identifies the item you are holding.
    /mi item repair Manually repairs the item you are holding.
    /mi item unidentify Manually UNidentifies the item you are holding.
    /mi item deconstruct Manually deconstructs the item you are holding.
    /mi give (player) (min-max) (unident-chance) (drop-chance) (soulbound-chance) (silent) Gives an item to a player
    ITEM MANAGEMENT
    /mi browse Pulls up an interactable library of ALL of our created items, sorted by item types. (Recommended for most item creation!)
    /mi itemlist Lists all items from a specific item type.
    /mi allitems Lists all created items, not as useful as /mi browse.
    /mi giveall Gives an item to all online players.
    CRAFTING STATIONS
    /mi stations list Lists all current crafting stations.
    /mi stations open (player) This opens the specified station, for the specified player. This is the command you would use to bind stations to blocks.
    UTILITY/OTHER
    /mi ability (player) (mod1) (val1) (mod2) (val2) Cast an MMOItems ability (works with MM skills as well). There is arguments to select the ability, the player who casts it, and modify the values like damage and cooldown etc.
    /mi heal Similar to essentials/CMI heal, this heals you and removes all negative effects.
    /mi debug info (player) Opens up a useful chat menu with information about the player. This includes stats like class and mana (if applicable).
    /mi list (type/spirit/ability) Shows all available types for weapons,staff spirits, entities etc.
    /mmotempstat (player) (stat name) (amount) (duration) Gives a temporary stat-boost to the player for x ticks. Stat names are in caps and underscores.
    GENERAL
    /mmoitems reload Reloads the entire plugin, after editing config files. No need to restart your server!
    /mi reload (adv-recipes/stations) Reloads the plugin or reloads the advanced recipes/crafting stations if applicable.
    ITEM GENERATOR
    /mi generate (extra-args) Check Obtaining an item for more info
