    编辑界面

    Revision ID

    Note

    • Revision ID
    • The Revision ID is used to determine if an item is outdated or not. You should incerease this whenever you make changes to your item!
    • 修改版 ID
    • 修订版 ID 用于确定项目是否过时。每当您对项目进行更改时，您都应该意识到这一点！

    Material

    Note

    • Material
    • Your item material
    • 材质
    • 修改物品的显示的图标内容

    Item Damage

    Note

    • Item Damge
    • Default item damage. This does NOT impact the item's max durability

    Custon Model Data

    Note

    • Custon Model Data
    • Your 1.14+ model data.

    Maximum Durability

    Note

    • Maximum Durability
    • The amount of uses before your item becomes unusable/breaks.
    • 设置物品的耐久上限

    Lost when Broken

    Note

    • Lost when Broken
    • if set to true,the item will be lost once it reaches 0 durablilty.
    • 该内容是设置当物品耐久使用完后,破坏该物品

    Display Name

    Note

    • Display Name
    • The item display name.
    • 设置显示名称

    Lore

    Note

    • Lore
    • The item lore.
    • 设置Lore

    NBT Tags

    Note

    • NBT Tags
    • Custom NBT Tags.
    • 自定义 NBT数据标签

    Displayed Type

    Note

    • Displayed Type
    • This option will only affect the type displayed on the item lore.
    • 用于显示该物品 属于什么类型(例如: 武器、饰品、消耗品等)

    Lore Format

    Note

    • Lore Format
    • The lore format decides where each stat goes.
    • Formats can be configured in the lore-formats folder

    Enchantments

    Note

    • Enchantments
    • The item enchants.
    • 设置附魔

    Hide Enchantments

    Note

    • Hide Enchantments
    • Enable to completely hide your item enchants. You can still see the glowing effect.
    • 设置是否隐藏附魔显示.即使隐藏之后附魔效果依旧存在有效.

    Permission

    Note

    • Permission
    • The permission needed to use this item.
    • 设置使用该物品的权限,如果玩家无对应的权限就无法使用

    Itemn Particles

    Note

    • Item Particles
    • The particles displayed when holding/wearing your item.
    • 装备佩戴该物品的显示粒子效果

    Disable Interaction

    Note

    • Disable Interaction
    • Disable any unwanted interaction:
    • block placement, item use...
    • 是否禁用交互(例如:方块与一些物品可以右键使用)

    Disable Crafting

    Note

    • Disable Crafting
    • Players can't use this item while crafting.
    • 是否禁用该物品参与制作合成

    Disable Smelting

    Note

    • Disable Smelting
    • Players can't use this item in furnaces.
    • 是否禁用该物品参与冶炼熔炉

    Disable Smithing

    Note

    • Disable Smithing
    • Players can't smith this item in smithing tables.
    • 是否禁用该物品参与铁砧的制作

    Disable Enchanting

    Note

    • Disable Enchanting
    • Players can't enchant this item.
    • 是否禁用该物品进行附魔

    Disable Repairing

    Note

    • Disable Repairing
    • Players can't use this item in anvils.
    • 是否禁用该物品通过铁砧修复

    Required Level

    Note

    • Required Level
    • The level your item needs in order to be used.
    • 使用该物品玩家需要的等级

    Required Class

    Note

    • Required Class
    • The class you need to profess to use your item.
    • 使用该物品需要的职业

    Critical Strike Chance

    Note

    • Critical Strike Chance
    • Critical Strikes deal more damage in % chance.
    • 设置暴击几率

    Critical Strike Power

    Note

    • Critical Strike Power
    • The extra damage weapon crits deals.(Stacks with default value) in %.
    • 设置暴击伤害

    Commands

    Note

    • Commands
    • The commands your item performs when right clicked.
    • use:
    • Write in the chat the command you want to add
    • To add a delay, use -d:
    • To make the command cast itself w/ console, use -c
    • To make the command cast w/ OP perms, use -op
    • 使用该物品执行命令,其中添加命令的时候 可以选择 后台执行(console,简写-c)或者用OP身份执行(-op)
    • '-op fly' - 使用OP身份进行执行,即使自身没有权限
    • '-c fly' - 通过后台执行
    • 区别 op 身份可执行一些需要玩家来执行的命令(不支持占位符)
    • 而 console 后台执行,则不支持执行玩家执行类型的命令.但是支持占位符
