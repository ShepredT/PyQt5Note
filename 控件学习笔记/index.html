    初步认识常用控件

    基础控件

    按钮

    Note

    可以被用户点击的共同性，在具体功能上有细微的差别

    • QpushButton - 按钮

    • QCommandLinkButton - 链接按钮

    • QRadioButton - 单选框

    • QCheckBox - 复选框

    输入控件

    Note

    • 用于接收用户输入的信息
    纯键盘输入

    Tip

    • QLineEdit - 单行输入框
    • QTextEdit - 多行输入框(可以放入图片，超链接等)
    • QPlainTextEdit - 普通的多行输入框(不支持，图片，超链接)
    • QKeySequenceEdit - 收集快捷方式的
    步长调节(QAbstractSpinBox)(键盘+鼠标)

    Tip

    • QDateTimeEdit - 采集日期与时间
      • QDateEdit - 单独日期调节
      • QTimeEdit - 单独时间调节
    • QSpinBox - 数字步长调节
    • QDoubleSpinBox - 浮点数调节
    组合框(下拉选项输入)

    Tip

    • QComboBox - 下拉选项框
      • QFontComboBox - 下拉式选项字体框
    滑块(QAbstractSlider)(鼠标)

    Note

    • QDial - 旋钮
    • QSlider - 滑块
    • QScrollBar - 滚动条
    橡皮筋选中

    Tip

    • QRubberBand
    对话框(QDialog)

    Tip

    • QColorDialog - 颜色对话框
    • QFileDialog - 文件对话框
    • QFontDialog - 字体对话框
    • QInputDialog - 输入对话框
    日期

    Tip

    • QCalendarWidget
